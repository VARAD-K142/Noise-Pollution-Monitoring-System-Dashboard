IoT Urban Noise Pollution Monitoring System
Public cloud dashboard using Firebase Realtime Database and ThingSpeak
This dashboard displays real-time and historical noise monitoring data collected using an ESP32 and MAX4466 acoustic sensor. The system processes sound signals into RMS and approximate dB values, classifies them into SAFE, HIGH, and HARMFUL levels, stores the data in Firebase, and visualizes trends through ThingSpeak graphs.

Noise Classification Thresholds
SAFE
Noise level is below 62 dB and is considered acceptable.
HIGH
Noise level is between 62 dB and 78 dB and should be monitored.
HARMFUL
Noise level is 78 dB or above and may require alerts or preventive action.
